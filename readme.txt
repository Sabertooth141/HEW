---------------------------------------------------------
リワインドプロトコル
---------------------------------------------------------
作者：丁 子豪（ディン・ズーハオ）

---------------------------------------------------------
ゲーム概要
---------------------------------------------------------
プレイヤーキャラクターがタイムストップ、タイムリワインドというスキルを持っている横スクロールアクションゲームです。
コンボアタックをできて、敵を倒しステージをクリアするゲームを目指しています。

---------------------------------------------------------
操作説明：
---------------------------------------------------------
本ゲームはキーボードとゲームパッド２つの操作手段利用できます。
キーボード：
    ゲーム画面：
        A/ D - 左右移動
        Space/ W - ジャンプ
        J - 攻撃
        Shift - リワインド
        K - 時間停止
        Esc - ゲーム終了
    メニュー画面：
        WASD/ 矢印キー - 選択
        Space / Enter - 確定
ゲームパッド：
    ゲーム画面：
        LS - 左右移動
        A - ジャンプ
        X - 攻撃
        LB - リワインド
        RB - 時間停止
        Back - ゲーム終了
    メニュー画面：
        十字キー - 選択
        A - 確定

---------------------------------------------------------
プレイヤースキル紹介：
---------------------------------------------------------
タイムストップ：世界を3秒程度プレイヤーだけ動けるようにする（途中でキャンセルできる）
タイムリワインド：プレイヤーが数秒前の自分の姿に戻れる、スキルキーを長押し、戻る時点を選んで、HPもその時点のHPに回復できる
    もしレッドフレーム（敵と一番近い残像）に戻れば、敵へ大きいダメージを与えるアタックができる

プレイヤー攻撃：
プレイヤーキャラが最大3段までの攻撃ができます。
１、２段目はライトアタックで、３段目は前に飛び出す突進攻撃であります

プレイヤー移動：
左右移動／ジャンプできます

敵タイプ：
本ゲームは2つタイプの敵があります
地雷：
    プレイヤーに突っ込むとき一定時間のチャージ後爆発する
UGV：
    無人戦車であり、プレイヤーに近づけばプレイヤーをノックバックする近距離攻撃する

---------------------------------------------------------
工夫した点／アピールポイント
---------------------------------------------------------
工夫した点はプレイヤーのコンボシステムと時間に関するシステムですね。
本ゲームは全てのアセット（攻撃SFX以外）全部自分で用意しました
    めっちゃしんどかったです…楽しく遊んでいただければ何よりです。

プレイヤーキャラクターの技はこのゲームの一番楽しいとごろだと思います。
    リワインドと普通攻撃合わせて敵を倒しましょー
